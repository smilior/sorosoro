01 要件定義 · 02/04

ペルソナ・ユーザーストーリー

おうち掃除ログ（sorosoro）の利用者像と、その利用者が求める行動を定義する。

確定メモ

ペルソナ（P-01〜P-03）およびユーザーストーリー（US-00〜US-10）はプロダクトブリーフに基づき確定済み。

MVP 対象は US-00〜US-09（Google ログイン・1 タップ記録・タスク CRUD・履歴・クラウド永続化・ダークモード含む）。US-10（世帯共有）は後続リリース。

ストーリー形式は「〜として〜したい。〜のため」を維持する。

ペルソナ

ID ペルソナ名 属性 困りごと ニーズ
P-01みさき30 代・共働き / スマホ中心 / 家事時間に余裕なし「シーツいつ洗った？」が分からず罪悪感が残る責められず、すぐ記録できる
P-02けんじ40 代・単身 / 掃除は溜めがち / スマホ操作は問題なし周期が分からず放置してしまう経過日だけ淡々と見える
P-03あやか20 代・同棲 / ペアで家事分担 / 個人メモ感覚を好む相手に責められたくない記録が欲しい個人メモ感覚で使える

ユーザーストーリー

ID ペルソナ ストーリー 優先度 MVP対象
US-00P-01みさきとして、Google でログインし自分のデータだけ見たい。端末を変えても同じ記録を使うため高○
US-01P-01みさきとして、ホームで「今日やった」を 1 タップで記録したい。手間を減らすため高○
US-02P-01みさきとして、タスクを名前と目安周期で登録したい。気になる掃除を残すため高○
US-03P-02けんじとして、前回から何日経ったか一覧で見たい。状況を把握するため高○
US-04P-01みさきとして、誤記録をアンドゥしたい。気軽に押せるようにするため高○
US-05P-02けんじとして、過去の日付でも記録したい。まとめて思い出したときのため高○
US-06P-03あやかとして、履歴を見て前回の実施日を確認したい。分担の目安を知るため中○
US-07P-01みさきとして、タスク名・周期を後から変えたい。生活リズムが変わっても使えるため中○
US-08P-02けんじとして、不要なタスクを履歴ごと消したい。リストをすっきり保つため中○
US-09P-01みさきとして、ダークモードで夜間も見やすくしたい。寝る前の確認を楽にするため低○（実装済）
US-10P-03あやかとして、世帯でタスクを共有したい。ペアで同じリストを見られるため低－

リリーススライス

リリース 含むストーリー 目的
第1リリース(MVP)US-00, US-01, US-02, US-03, US-04, US-05, US-06, US-07, US-08, US-09Google ログイン・1 タップ記録・経過日一覧・タスク CRUD・履歴・Turso へのユーザー単位永続化・ダークモードで「責めない掃除メモ」の価値を検証
第2リリース以降US-10世帯共有（OUT-01 の解除候補）。複数人での同一タスク編集は当面スコープ外
