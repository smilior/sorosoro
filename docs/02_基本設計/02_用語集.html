用語集（ユビキタス言語）

おうち掃除ログのドメイン用語とインフラ用語を統一する。日本語名・英語名・コード上の命名・定義を1対1で対応させる。

確定メモ

ドメインの正は docs/_spec/PRODUCT_BRIEF.md と src/lib/domain/task.ts。DB カラムは snake_case（schema.ts）、TypeScript プロパティは camelCase。

種別 規約 例
論理エンティティ英語 PascalCaseTask, TaskLog, User

TS プロパティcamelCasecycleDays, userId, doneDate
実施日ISO 日付文字列"YYYY-MM-DD"
派生値永続化しない。都度計算ratio, elapsedDays, state
