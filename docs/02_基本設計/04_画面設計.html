02 基本設計 · 04/07

画面設計書

おうち掃除ログの画面構成・遷移・レイアウト・デザイントークンを定義する。モバイルファースト Web。アプリ画面は認証必須。

確定メモ

ハイファイ参考は prototype/。本番は Next.js の App Router（/login, /home 等）。デザイントークンは PRODUCT_BRIEF と一致させる。

画面遷移図

S-00 ログイン（未認証）
  └─ Google ログイン成功 → S-01 ホーム

S-01 ホーム（要認証）
  ├─ ＋ → S-02 タスク登録
  ├─ カード本体タップ → S-03 履歴
  │     ├─ 編集 → S-02 タスク編集
  │     └─ 日付を選んで記録 → S-04 記録シート
  ├─ 「今日やった」タップ → 記録（画面遷移なし）
  ├─ 「今日やった」長押し 480ms → S-04 記録シート
  └─ ログアウト → S-00

S-02 保存 → 戻り先（home または history）
S-02 削除確定 → S-01 ホーム
S-04 記録/キャンセル → 下に閉じる（オーバーレイ）

未認証で /home → Middleware が /login へ

トークン 値 用途

  • フォント: Zen Maru Gothic 400/500/700（Google Fonts）
  • タップ領域 ≥ 44px、カード角丸 20px、シャドウ 0 1px 3px rgba(80,70,55,.06)
  • アイコンはインライン SVG（チェック・プラス・シェブロン）のみ。絵文字不使用
  • ダークモードは実装済み（US-09）。トークンはライトを基本仕様とし、ダークは同等トーンで反転

画面詳細

S-00 ログイン

  • パス: /login
  • 目的: Google でサインイン
  • 表示条件: 未認証。Cookie ありなら /home へリダイレクト
領域要素表示条件・挙動
ヒーローアプリ名・一言説明常時
CTA「Google で続ける」ボタンBetter Auth social sign-in

S-01 ホーム

  • パス: /home
  • 目的: 状況把握と1タップ記録
  • 表示条件: 認証必須
領域要素表示条件・挙動
ヘッダーアプリ名 12px/700 sub + 今日日付 22px/700常時
ヘッダー右＋ボタン 44×44 角丸16px accent-soft→ S-02 新規
リストタスクカード（gap 10px、左右 padding 16px）ratio ソート。カードタップ → S-03
カード左タスク名 16px/700 + メタ 12.5px経過日のみ強調。未記録は「まだ記録なし」
カード右「今日やった」ボタン ≥44px 角丸14pxタップで記録。長押しで S-04。当日済は「記録済み」
リスト末尾長押しヒント 11.5px sub 中央タスクがあるとき
空状態登録を促す文言タスク0件（シード前）

S-02 タスク登録/編集

  • 目的: タスク名・周期の登録・変更・削除
  • 表示条件: 認証必須
領域要素表示条件・挙動
ヘッダー戻る 38×38 + タイトル 18px/700「タスクを追加」/「タスクを編集」
フォームタスク名入力 角丸16px font-size 16pxfocus border accent
周期プリセット 7/14/30 日 + 数値「◯日ごと」1–365選択中は accent-soft
補足目安である旨 12px sub常時
保存全幅ボタン accent名前空/周期不正で disabled
削除「このタスクを削除」del 色 → 二段確認編集時のみ。confirm() 不使用

S-03 履歴

  • 目的: サマリーと実施履歴の確認、過去日記録導線
  • 表示条件: 認証必須・対象タスクが自分のもの
領域要素表示条件・挙動
ヘッダー戻る + タスク名省略 + 「編集」編集 → S-02
サマリー3カラム: 目安周期 / 前回 / 次の目安「ごろ」常時
アクション「＋ 日付を選んで記録」全幅→ S-04
履歴リスト1カード内の行区切り。日付 + 相対ラベル年跨ぎのみ年表示。空状態あり

S-04 記録シート

  • パス: オーバーレイ
  • 目的: 日付を選んで記録
  • 表示条件: ホーム長押しまたは履歴から起動（要認証）
領域要素表示条件・挙動
スクリムrgba(42,36,28,.34)タップで閉じる
シート角丸28px、グラバー、下から 0.28s スライド左右下 8px 浮き
タイトル「日付を選んで記録」+ タスク名常時
クイック今日 / 昨日 / おとといタップで即記録
任意日date input max=今日 + 「この日で記録」未来不可
キャンセルテキストボタン閉じる
