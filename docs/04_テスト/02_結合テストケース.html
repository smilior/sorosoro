結合テストケース

モジュール・コンポーネント間のインターフェース結合を検証するケースを定義する。基本設計に対応（V字）。

記入ガイド

{{ }} を自アプリの値に置換。(例)行は削除。ケースは行を増やす。

基本設計（外部仕様）に対応。モジュール間・API・DB連携の結合を STG で確認する。

結合の観点（画面→API→DB、認証連携、外部サービス連携）を先に洗い出す。

不合格ケースは起票し、解消まで追跡する。

対象インターフェース

区分 結合対象
画面-API 画面操作から API 呼び出し・応答表示までの経路
API-DB API から永続化・整合性制約までの経路
認証連携 ログイン・セッション・認可判定の連携
